输入URL到页面渲染的过程 2019-05-23

输入 URL 到页面渲染中间发生了什么？

WEB 应用的生命周期（简述）

  1. 用户 输入 URL 或者点击链接（DNS 查询）
  2. 浏览器 生成请求并发送至服务器 (TCP 三次握手四次挥手)
  3. 服务器 执行某些动作或者获取某些资源；将响应发送至客户端
  4. 浏览器 处理 Html、CSS、和 JavaScript 并构建结果页面
  5. 浏览器 监控事件队列 处理事件
  6. 用户 用户与 web 应用进行交互 期间一直重复 5、6 阶段
  7. 用户 关闭 web 应用服务
  8. 生命周期结束
